## Pré-Requisitos
* IDE: Netbeans 16 - https://archive.apache.org/dist/netbeans/netbeans-installers/16/Apache-NetBeans-16-bin-windows-x64.exe
* Mysql 8.0.XX Community Server - https://dev.mysql.com/downloads/file/?id=545896
* Executar script DumpBancoPDV.sql no seu Mysql para criar a base exemplopdv e as tabelas
